# Prediect-Salary-Using-Linear-Regression
Salary prediction using Linear Regression is a method in Machine Learning that estimates a person’s salary based on factors like experience and skills.

It works by finding a linear relationship between input features and salary, typically using the equation y = mx + b. Once trained on historical data, the model can predict salary for new inputs.

This approach is simple, fast, and widely used for basic salary estimation tasks.
